# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **React.js** to showcase my projects, skills, and contact information.

The website highlights my work in **AI, Data Analysis, and Web Development**.

🔗 Live Website:  
https://anupam2090.github.io/anupam-portfolio

## 🚀 Features

- Modern responsive portfolio design
- Animated gradient background
- Project showcase section
- Contact information section
- GitHub and LinkedIn links
- Resume download option
- Smooth scrolling navigation

## 💻 Installation & Running Locally

Clone the repository:


git clone https://github.com/Anupam2090/anupam-portfolio.git


Navigate to the project folder:


cd anupam-portfolio


Install dependencies:


npm install


Run the project:


npm start


The website will open at:


http://localhost:3000


---

## 🌍 Deployment

This project is deployed using **GitHub Pages**.

To deploy:


npm run deploy
